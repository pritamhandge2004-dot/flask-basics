Multiple URL Parameters:

This route captures two parameters from the URL:

  1. category_name (string): "{{ category_name }}"
  2. product_id (integer): {{ product_id }}
@app.route('/category/<category_name>/product/<int:product_id>')
def category_product(category_name, product_id):
    # Access both parameters in the function
    return render_template('category_product.html',
        category_name=category_name,
        product_id=product_id)
